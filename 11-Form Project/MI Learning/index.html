MACHINE LEARNING НА PYTHON

Що означає Machine Learning?

Машинне навчання (МО, Machine Learning, ML) — великий підрозділ штучного інтелекту, що вивчає методи побудови алгоритмів, здатних навчатися.

Для чого потрібен Machine Learning

Загальну мету machine learning можна охарактеризувати як автоматизацію та оптимізацію рішень складних рутинних задач у найрізноманітніших сферах, від метеорології до комунікацій (про котрі піде мова нижче). Сучасний фунціонал інструментів машинного навчання справді вражає: додатки вже здатні розпізнавати мову, жести та образи, проводити медичну та технічну діагностики, біржовий і фінансовий аналізи, систематизувати документацію та виявляти спам.

Головна ціль

При вмілому підході, комбінуючи різні види машинного навчання, можна досягти автоматизації більшості рутинних бізнес-процесів. Іншими словами, алгоритми та роботи, підготовлені за допомогою машинного навчання, можуть виконувати всю рутинну роботу. Людям же залишається вся творча частина: складання стратегій, ведення переговорів, укладання договорів та інше. Це важливий фактор, оскільки машина не може вийти за задані рамки, а людський мозок здатний мислити нешаблонно.
